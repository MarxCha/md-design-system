# Prompts de imagen — kit Deloitte-MD (Gemini / Nano Banana Pro)

Reglas comunes (el duotono azul MD se aplica encima por CSS):
- Foto **realista, editorial**, NO abstracta.
- Tonos **fríos y desaturados** (azul pizarra, gris); el color lo pone el overlay azul MD.
- **Sin texto legible** en la imagen (el texto lo pone la maqueta).
- Composición limpia, con **espacio negativo**; sujeto descentrado.
- Toques sutiles de verde esmeralda/teal OK; nada saturado.

Dónde va cada archivo (deja el .jpg con ese nombre y vuelve a renderizar con `render.sh`):
- `examples/sener/assets/cover.jpg`         → portada SENER (banda hero)
- `examples/sener/assets/opener-galera.jpg` → divisor GALERA (página completa)
