El reto: un libro de 300 páginas.

Tablas extensas, cuadros, diagramas y gráficas — maquetados a escala, con calidad medible.

Una obra técnica de 300 páginas no es un problema de redacción, sino de maquetación a escala y de control de calidad sostenido página por página: viudas, huérfanas, tablas partidas y desbordes se acumulan en obras largas.

MD lo resuelve con dos fortalezas. Primero, una trayectoria editorial verificable: Libros Blancos y memorias documentales entregados a dependencias federales y estatales, incluido el sector energético.

Segundo, GALERA, un motor propio que compone la maqueta de forma programática y la audita de forma determinista, corrigiendo en la fuente hasta converger. La calidad se mide, no se opina.

Capacidades editoriales|GALERA · cómo funciona

02 · Cómo funciona

Maquetación con control de calidad medible.

Cuatro pasos, una regla de oro: ninguna corrección se conserva si no baja el conteo de defectos.

Paso 01

Composición programática

Texto, datos y tablas se componen con Typst: portada, índice, folios, figuras y gráficas.

Paso 02

Auditoría determinista

Se mide la posición de cada elemento: desbordes, tablas cortadas, viudas y huérfanas.

Paso 03

Auto-sanación

ReLook corrige la fuente y recompila; conserva el cambio solo si bajan los defectos.

Paso 04

Escape a revisión

Si una página no converge, se marca para revisión humana. Nunca se “maquilla” un defecto.
